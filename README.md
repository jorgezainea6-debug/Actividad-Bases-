# Student Database README

## Description

This project contains a simple database designed to store basic information about university students. The database focuses on three main attributes:

* **Name**: The full name of the student.
* **Age**: The student's age.
* **Semester**: The current semester the student is enrolled in at the university.

## Purpose

The purpose of this database is to provide a minimal and structured way to manage student information. It can be used for educational projects, small applications, or as a starting point for more complex student management systems.

## Structure

Each record in the database represents a single student and includes the following fields:

| Field    | Type    | Description                    |
| -------- | ------- | ------------------------------ |
| Name     | String  | Student's full name            |
| Age      | Integer | Student's age in years         |
| Semester | Integer | Current semester in university |

## Example Entry

```
Name: Valeria
Age: 20
Semester: 4
```

## Usage

This database can be implemented using any database management system (e.g., MySQL, PostgreSQL, SQLite) or even as a simple data structure in a programming language.

## Notes

* The database is intentionally simple and can be extended with additional fields such as student ID, major, or GPA.
* Ensure data validation is applied when inserting records (e.g., age should be a positive number, semester should be within a valid range).
